# Laptop Web App

Demo web thương mại điện tử, sản phẩm là laptop, gồm đầy đủ các chức năng của một web thương mại điện tử hiện nay

## Kiến trúc
- Dùng kiến trúc Layered Architecture

## Công nghệ
- Java 8
- Spring boot
- Spring Mail
- Spring Sercuriry
- Spring JPA
- Jsp
- Servlet
- PostgreSQL
- Cloudinary
- PayPal
- Login Google, Facebook
- AJAX
- Hibernate
- Chart.js
